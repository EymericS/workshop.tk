# Roadmap of my coding journey

- install yarn
```shell script
npm install -g yarn
```
- init yarn
```shell script
yarn init -y
```
- parcel
    - install
    ```shell script
    yarn add -D parcel@next
    ```
    - command
    ```shell script
    npx parcel index.html   ## run dev server with auto reload
    npx parcel build index.html   ## build app to deploy
    ```
- install react and react-dom
```shell script
yarn add react react-dom
```
- install Ant Design
```shell script
yarn add antd
```
- git
    - init
    ```shell script
    git init
    ```
    - commande
    ```shell script
    git status
    git add .
    git commit
    git push
  
    git commit -am "message"
    ```
